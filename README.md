### What I learned

This was a great introduction to learning media queries in CSS!

```css
@media (max-width:375px) {
    body {
        background: url(./images/pattern-background-mobile.svg) no-repeat hsl(225, 100%, 94%);
    }

    .card{
        width: auto;
        margin: 20vh auto;
    }

}
```
